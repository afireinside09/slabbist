# Sub-project 8 — Analytics & reporting

Dashboards, exports, and the inventory story.

## Scope

- Store dashboard — daily/weekly/monthly buys and sells.
- P&L tracking per transaction and aggregate.
- Best/worst performing buys.
- Inventory mode — case vs. back stock tagging, location notes, aging report (what's been sitting too long).
- Employee performance metrics (scans, buys, margin per employee).
- Exports: CSV / Excel for accounting.
- Tax-ready transaction reports.
- End-of-day recap (builds on transaction log from sub-project 6).

## Features captured

- Store dashboard with daily/weekly/monthly buys and sells
- P&L tracking
- Best/worst performing buys
- Inventory aging reports
- Employee performance metrics
- Export to CSV/Excel
- Tax-ready reports
- End-of-day recap

## Dependencies

- Sub-project 6 (transactions)
- Sub-project 7 (margin rules for P&L calculations)
