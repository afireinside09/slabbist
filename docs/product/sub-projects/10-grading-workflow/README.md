# Sub-project 10 — Grading workflow

For stores that also submit cards for grading.

## Scope

- Submission prep tool — organize cards into PSA / BGS / CGC / SGC submissions with declared values, service levels, and printable manifests.
- Submission tracking — order numbers, status, ETA.
- ROI calculator once grades come back — compares actual grade revenue against projections.
- Historical submission performance — hit rates at each grader over time.

## Features captured

- Submission prep tool
- Submission tracking
- ROI calculator
- Historical submission performance

## Dependencies

- Sub-project 1 (auth, store)
- Sub-project 2 (cards for selecting submission candidates)
- Sub-project 4 (comp engine for declared value suggestions)

## Unblocks

- Nothing blocking.
