# Sub-project 11 — Integrations

Hooks into the broader hobby-store tech stack.

## Scope

- Square POS integration — push transactions into Square for checkout.
- Shopify integration — sync inventory into a Shopify storefront.
- eBay seller integration — pull active listings, auto-relist sold inventory.
- TCGplayer seller integration — inventory and pricing sync.
- QuickBooks / accounting export.
- Label printer support for inventory tags (Dymo, Brother, Zebra).
- Barcode / QR scanner hardware support.

## Features captured

- Square, Shopify, eBay seller, TCGplayer seller integrations
- QuickBooks export
- Label printer hardware
- Barcode / QR scanner hardware

## Dependencies

- Sub-project 6 (transactions to push)
- Sub-project 8 (exports build on analytics pipelines)

## Unblocks

- Nothing blocking — pure integration surface area.
